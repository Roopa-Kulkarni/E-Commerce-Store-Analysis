# E-Commerce-Store-Analysis

Objective

To analyze Olist's e-commerce sales and operational data to identify sales trends, customer purchasing behaviour, product performance, payment patterns, delivery efficiency, and factors influencing customer satisfaction, and provide actionable insights to improve business performance.

Tools used Excel| Tableau| Power BI

Key KPI'S

Weekday Vs Weekend (order_purchase_timestamp) Payment Statistics

Number of Orders with review score 5 and payment type as credit card.

Average number of days taken for order_delivered_customer_date for pet_shop

Average price and payment values from customers of sao paulo city

Relationship between shipping days (order_delivered_customer_date - order_purchase_timestamp) Vs review scores.
